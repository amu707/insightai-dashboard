# InsightAI

AI-powered business analytics dashboard built using Streamlit and Python.

## Features

* Upload CSV and Excel files
* Revenue Analysis
* Business Insights
* Top Performer Detection
* PDF Report Generation
* Interactive Data Visualization

## Tech Stack

* Python
* Pandas
* Streamlit
* Matplotlib
* ReportLab

* AI Chat with Business Data
* Revenue Forecasting
* Automated Executive Reports
* Business Alerts
* AGI Analyst Platform
